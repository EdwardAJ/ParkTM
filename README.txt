PENJELASAN APLIKASI

PARK-TM by Trustix (Kelompok 13)
Aplikasi untuk menggantikan karcis parkir di ITB.
Menggunakan Android Studio untuk UI dan Java untuk UX.
Mockup UI menggunakan www.marvelapp.com.

Folder ini terdiri dari file .apk di dalam folder "APLIKASI(apk)" dan sourcecodenya pada folder lainnya.
Aplikasi ini dibuat di Android Studio menggunakan Java.
Adapun halaman yang dicoding adalah homepage dan halaman kedua saja.
Namun halaman kedua hanya didukung oleh UI, sedangkan halaman pertama menggunakan UI dan UX (pada tombol "Daftar").

CARA MENGGUNAKAN APLIKASI

Cara Daftar : 
1. Buka Aplikasinya
2. Isi data form nya terlebih dahulu dengan jujur ya
3. Klik Daftar
4. Masukkan Serial Number RFID KTM
5. Ikuti Tutorialnya

Cara Login : (belum jalan)
1. Buka Aplikasinya
2. Isi data form nya terlebih dahulu dengan jujur ya
3. Klik Masuk

Cara Logout :(belum jalan)
1. Klik Logout

Fitur lain :
A. Lihat Transaksi (belum jalan)
  - Berisi data riwayat parkir yang terdiri dari tanggal masuk, tanggal keluar, waktu masuk dan waktu keluar
B. Status (belum jalan)
  - Berisi status dari motor yang didaftarkan
    > Belum Parkir
    > Lagi Parkir
    > Sudah Parkir
